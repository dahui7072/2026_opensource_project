# 2026_opensource_project

# 🛴 AI 스마트 모빌리티 위반 관제 시스템

> YOLOv8 기반 전동 킥보드 안전 수칙 위반 실시간 감지 및 통계 대시보드

---

## 📌 프로젝트 개요

전동 킥보드의 **헬멧 미착용**과 **2인 탑승** 위반을 실시간으로 감지하고,  
위반 현황을 웹 대시보드로 시각화하는 AI 기반 관제 시스템입니다.

경찰 인력 의존의 한계를 보완하고, 킥보드 관련 사고를 예방하는 것을 목표로 합니다.

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| AI 모델 | YOLOv8 (Ultralytics) |
| 객체 탐지 | OpenCV |
| 백엔드 | Flask |
| 프론트엔드 | HTML, CSS, Chart.js |
| 데이터 저장 | CSV |
| 버전 관리 | Git / GitHub |

## 📁 디렉터리 구조


## ⚙️ 실행 방법

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 서버 실행

```bash
python app.py
```

### 3. 대시보드 접속

브라우저에서 `http://127.0.0.1:5000` 접속

## 🔍 주요 기능

**실시간 감지**
- 웹캠 영상에서 킥보드, 사람, 헬멧 객체 탐지
- 헬멧 미착용 / 2인 탑승 위반 자동 판정
- 위반 시 화면에 경고 오버레이 표시

**통계 대시보드**
- 오늘 총 위반 건수 / 헬멧 미착용 수 / 2인 탑승 수 실시간 표시
- 시간대별 위반 건수 막대 차트
- 위반 유형 비율 도넛 차트
- 누적 위반 기록 실시간 로그 테이블 (5초 갱신)

## 📊 감지 클래스

| 클래스 | 설명 | 학습 방식 |
|--------|------|-----------|
| `helmet` | 헬멧 착용 | 직접 촬영 + 공개 데이터셋 |
| `no_helmet` | 헬멧 미착용 | 직접 촬영 + 공개 데이터셋 |
| `person` | 사람 | YOLOv8 사전학습 모델 활용 |
| `kickboard` | 킥보드 | 공개 데이터셋 활용 |

---

## 📝 위반 판정 기준

- **헬멧 미착용**: `no_helmet` 클래스 탐지 시 위반 판정
- **2인 탑승**: 킥보드 바운딩 박스 내 `person` 객체가 2개 이상일 경우 위반 판정

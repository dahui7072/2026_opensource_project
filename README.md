# 🛴 AI 스마트 모빌리티 위반 관제 시스템

> YOLOv8 기반 전동 킥보드 안전 수칙 위반 실시간 감지 및 통계 대시보드

---

## 📌 프로젝트 개요

전동 킥보드의 **헬멧 미착용**과 **2인 이상 탑승** 위반을 실시간으로 감지하고,
위반 현황을 웹 대시보드로 시각화하는 AI 기반 관제 시스템입니다.
경찰 인력 의존의 한계를 보완하고, 킥보드 관련 사고를 예방하는 것을 목표로 합니다.

---

## 🛠 기술 스택

| 분류 | 기술 |

## ⚙️ 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 서버 실행
```bash
python app.py
```

### 3. 대시보드 접속
브라우저에서 `http://127.0.0.1:5001` 접속

> 기본은 `samples/` 폴더의 영상 및 이미지 파일을 순서대로 순환 재생합니다. 웹캠으로 돌리려면 `app.py`의 `USE_WEBCAM = True`로 변경하세요.

## 🔍 주요 기능

**실시간 감지**
- 웹캠 또는 사전 녹화된 영상 파일에서 킥보드, 사람, 헬멧 객체 탐지 (`USE_WEBCAM` 토글로 전환)
- 헬멧 미착용 / 2인 이상 탑승 위반 자동 판정
- 위반 시 화면에 경고 오버레이 표시

**통계 대시보드**
- 오늘 총 위반 건수 / 헬멧 미착용 수 / 2인 탑승 수 실시간 표시
- 시간대별 위반 건수 막대 차트
- 위반 유형 비율 도넛 차트
- 누적 위반 기록 실시간 로그 테이블 (5초 갱신)

## 📊 감지 클래스

YOLOv8 모델이 직접 탐지하는 클래스는 아래 3개입니다. `no_helmet`은 별도 클래스가 아니라, 탐지 결과를 기반으로 `violation.py`에서 판정하는 **위반 유형**입니다.

| 클래스 | 설명 | 학습 방식 |
|--------|------|-----------|
| `helmet` | 헬멧 | 직접 촬영 + 공개 데이터셋 |
| `person` | 사람 | YOLOv8 사전학습 모델 활용 |
| `kickboard` | 킥보드 | 공개 데이터셋 활용 |

## 📝 위반 판정 기준

- **헬멧 미착용**: 킥보드 bbox와 겹치는 `person`(탑승자) 중, 그 사람의 bbox와 겹치는 `helmet`이 하나도 없는 경우. 최근 30프레임 중 절반 이상 이 상태가 지속될 때만 위반으로 확정합니다 (일시적 오탐 방지).
- **2인 이상 탑승**: 하나의 킥보드 bbox와 겹치는 `person`이 2명 이상인 경우. 마찬가지로 최근 30프레임 기준 누적 판정을 적용해 지나가는 행인이 우연히 같이 잡히는 오탐을 방지합니다.
